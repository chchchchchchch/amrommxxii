 HOWTO
==========================================================================

  _/      <- main working folder

- Please use this folder when you create files and folders
 (to keep our top level clean and comprehensive)
 -> _/2202251146_file.txt
 -> _/2202251223_folder
- Please use the following naming convention

  YEAR
  |   DAY
  |   |   MINUTE
  --  --  --
  --------------------------
  2202251147_description.ext
  --------------------------
    --  --
    |   HOUR
    MONTH

- create a folder if your working structure
  requires more complex hierarchies e.g. processing sketches.
  file naming and structurte in subfolders is up to
  but for compatibility reason please stick to ascii:
  _/2202251216_ch/mytotallycrazyfilename.jpg
  _/2202251216_ch/subfolder/subfolder/README.txt

- lib/      <- library: reusable and shared.
- utils/    <- helpful stuff. reusable and sort of stable.
